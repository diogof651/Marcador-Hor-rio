Relacionamentos:
Uma Empresa pode ter vários Chamados.
Um Meio de Comunicação pode ser usado em vários Chamados.
Um Chamado pode estar associado a uma Empresa, um Meio de Comunicação e, possivelmente, a uma Compilação.
